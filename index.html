Alignment

You can also use flexbox features to align flex items along the main or cross axis. justify-content controls where the flex items sit on the main axis and it accepts six different values flex-start (default): items are packed toward the start line flex-end: cross-end margin edge of the items is placed on the cross-end line center: items are centered along the line space-between: items are evenly distributed in the line; first item is on the start line, last item on the end line space-around: items are evenly distributed in the line with equal space around them space-evenly: items are distributed so that the spacing between any two adjacent alignment subjects, before the first alignment subject, and after the last alignment subject is the same Use the following demo to see them in action:
